웹 접근성 검사 도구

  • N-WAX 웹 접근성을 빠르고 쉽게 평가할 수 있도록 제작한 도구다. 파이어폭스와 크롬 브라우저의 확장 프로그램으로 제공된다.
  • UIA Varify 플래시, 실버라이트 등 애플리케이션의 접근성 정보를 탐색할 수 있는 도구다.
  • PEAT 웹 콘텐츠를 포함해서 스크린에 나타나는 대상이 발작 위험을 일으키는지 식별하는 도구다.
  • Contrast Ratio 포토샵에서 전경색과 배경색의 명도 차이를 비교할 수 있게 도와주는 확장 프로그램이다.
  • Screen Readers 시각 장애인이 컴퓨터를 사용할 때 화면에 표시되는 입출력 정보를 음성으로 알려주는 프로그램이다.
  • 웹 브라우저 개발자 도구 웹 콘텐츠 구성 요소를 탐색할 수 있으여 인터넷 익스플로러 외에도 대부분의 브라우저가 지원한다.
